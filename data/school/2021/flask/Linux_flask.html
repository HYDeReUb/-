Flask安裝(Linux)

參考資料:https://linuxize.com/post/how-to-install-flask-on-ubuntu-18-04/

1.安裝venv
sudo apt install python3-venv

2.新增&設定位置
mkdir my_flask_app
cd my_flask_app

3.檢查
python3 -m venv venv

4.連結venv
source venv/bin/activate

5.安裝flask
pip install Flask

6.取消連結venv
deactivate
